
## Сущность профиля

Описание:
Содержит информацию о профиле пользователя, типы, селекторы для получения данных из состояния.

#### Public api

- Components

`ProfileCard` - компонент с информацией по профилю пользователя

- types

`Profile` - Тип, описывающий профиль пользователя

- selectors

`getProfileData` - Селектор для получения информации о профиле пользователя
